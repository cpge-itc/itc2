def moyenne(df, col):
    m = 0
    for i in df.index:
        m += df.loc[i, col]
    return m / len(df)

Question

√âcrire une fonction ecart_type(df, c) qui renvoie l‚Äô√©cart-type des valeurs de la colonne c du dataframe df. On rappelle que l‚Äô√©cart-type d‚Äôune s√©rie de valeurs \(x_1, \ldots, x_n\) est donn√© par :

\[\sqrt{\frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})^2}\]

o√π \(\bar{x}\) est la moyenne des valeurs \(x_1, ..., x_n\).

Remarque : On √©vitera de calculer plusieurs fois la m√™me moyenne.

Solution

def ecart_type(df, col):
    m = moyenne(df, col)
    s = 0
    for i in df.index:
        s += (df.loc[i, col] - m)**2
    return (s / len(df))**0.5

On remarque que les attributs sont sur des √©chelles tr√®s diff√©rentes (par exemple, l‚Äô√¢ge est entre 0 et 80, alors que la classe du billet est entre 1 et 3).

def distance(p1, p2):
    d = 0
    for c in ["Pclass", "Sex", "Age", "Fare"]:

def plus_frequent(L): # renvoie la classe qui appara√Æt le plus souvent dans L
    compte = {}
    for e in L:
        compte[e] = compte.get(e, 0) + 1
    return max(compte, key=compte.get)
